## 구성

- **10종 코인**: BTC · ETH · SOL · TRX · HYPE · LINK · BNB · XRP · AAVE · DOGE
- **130여 지표** — 21개 탭으로 분류 (개요, 시장동역학, 온체인, 홀더, 밸류에이션, 수익성/성장, 리스크 조정, 유동성 품질 등)
- **Mock 데이터** (실 API 연동 전, 합리적 추정치)
- **실제 자동 수집 가능한 영역만 포함** (수작업/정성평가 제외)

## 특징

- 시총 순위 기반 sticky 컬럼 (TOP3 하이라이트)
- 섹터 필터 + 컬럼 정렬
- 모든 헤더 호버 시 지표 설명 툴팁
- F&G Index / BTC Dominance / 스테이블 MC 매크로 배너
